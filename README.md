# Visare Augenheilkunde

Projeto de website moderno para clínica de oftalmologia em Friedrichshafen.

## Tecnologias

Este projeto foi desenvolvido com:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Configuração do Ambiente

Requisitos: Node.js e npm instalados - [instalar com nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

```sh
# Instalar dependências
npm i

# Iniciar servidor de desenvolvimento
npm run dev

# Build para produção
npm run build

# Executar testes
npm test

# Preview da build de produção
npm preview
```

## Scripts Disponíveis

- `npm run dev` - Inicia o servidor de desenvolvimento
- `npm run build` - Cria build de produção
- `npm run build:dev` - Cria build em modo desenvolvimento
- `npm run lint` - Executa linter
- `npm run preview` - Preview da build de produção
- `npm test` - Executa testes
- `npm test:watch` - Executa testes em modo watch

## Estrutura do Projeto

- `/src` - Código fonte da aplicação
- `/public` - Assets estáticos
- `/src/components` - Componentes React
- `/src/pages` - Páginas da aplicação
